# Электронный дневник для школы

**Дипломный проект** по теме: *«Разработка электронного дневника для школы»*.

Веб-приложение для учёта оценок, расписания, домашних заданий и пропусков с разделением ролей: ученик, учитель, администратор.

## Запуск

```bash
npm install
npm start
```

Откройте в браузере: **http://localhost:3000**

## Вход в систему

На странице входа выберите **категорию** и введите **пароль**:

| Категория | Пароль |
|-----------|--------|
| Учитель   | `111`  |
| Ученик    | `222`  |
| Родитель  | `333`  |

## Возможности системы

- **Учитель** — выставляет и удаляет оценки, добавляет, изменяет и удаляет уроки в расписании
- **Ученик** — только просмотр своих оценок и расписания класса 10А
- **Родитель** — только просмотр оценок и расписания ребёнка (Иван Петров)

## Для пояснительной записки

1. **Актуальность** — цифровизация образования, отказ от бумажных дневников.
2. **Цель** — разработать веб-систему электронного дневника.
3. **Задачи** — анализ требований, проектирование, реализация, тестирование.
4. **Средства** — Node.js, Express, EJS, файловое хранилище.
5. **Развитие** — миграция на PostgreSQL, API для мобильного клиента, HTTPS.
